== Description ==

Automatically add Hyves respect buttons to your posts. It works for single pages and loops and is display options are customizable.

 ** Please deactivate this plugin, the Hyves social network has been shut down**
 More information can be found <a target="_blank" href="http://www.telecompaper.com/news/tmg-to-shut-down-hyves-report--976728">here</a>.

 We would like to thank everyone for using this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and move the `hyves-respect` folder to your `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. After activation you can configure the plugin using the `settings` link or the menu
